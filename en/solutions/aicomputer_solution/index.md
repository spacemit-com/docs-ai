# AI Computer Solution Overview

## Zenow

**Zenow** is a locally deployed AI knowledge assistant desktop application. All data processing is performed locally to protect privacy and data security. It supports multi-model management, intelligent conversation, knowledge base question answering, voice interaction, and related capabilities.

- [Zenow](zenow.md)

## Yumeet

**Yumeet** is a locally deployed AI meeting assistant desktop application. Audio processing, transcription, translation, and summarization are all performed locally to improve meeting documentation efficiency while protecting private data. Built on a speech pipeline that includes VAD, ASR, diarization, translation, and TTS, together with LLM-based summarization, it supports real-time recording, offline audio import, meeting archiving, and retrieval.

- [Yumeet](yumeet.md)

## Seewise

**Seewise** is an intelligent video search engine that supports local video upload and RTSP camera connectivity. It automatically analyzes video content, enabling rapid retrieval of target video segments through natural language queries.

- [Seewise](seewise.md)

## Agentforce

**Agentforce** is an AI digital employee management platform built on Hermes Agent and OpenClaw. It provides a visual web interface for creating, managing, and interacting with multiple AI digital employees. Each digital employee has an independent personality, memory, and skill set, and can learn autonomously and execute tasks.

- [Agentforce](agentforce.md)

## SpacemiT AI Lab

**SpacemiT AI Lab** is a web-based AI evaluation platform. Cloud-based K3 instances can be requested through the web interface, making it possible to evaluate model inference results and real performance data on the SpacemiT K3 AI CPU online without any hardware preparation, enabling a **zero-configuration, ready-to-use** evaluation experience.

- [SpacemiT AI Lab](ailab.md)

## Claude Code

**Claude Code** is a terminal-based AI coding assistant developed by Anthropic. It can run on the SpacemiT K-series platform to support software development workflows.

- [Claude Code](claude.md)

## OpenClaw

**OpenClaw** is an open-source AI agent developed by Austrian programmer Peter Steinberger and released in November 2025. It can run on the SpacemiT K-series platform.

- [OpenClaw](openclaw.md)

## Hermes

**Hermes** is an open-source autonomous AI agent developed by Nous Research and released as open source in February 2026. It can run on the SpacemiT K-series platform.

- [Hermes](hermes.md)
